# Course-Management
    - As a user, I want to be able to manage my courses for the current semester.
    - Students can add their personal information, find all courses and find courses by instructor. 
    - The students information consists of first name, last name, course, student id, gpa and email address.
    - Students can find their course number, course title, instructor and semester for the course. 

# Technologies used 
    - Java
    - Spring Boot
    - MySQL
    - Swagger
    - Bootstrap
    - ReactJS
    - React Query

# Architecture
    - Edge Service
    - Config Server
    - Service Registry
    - Crud Service Student API
    - Crud Service Course API
    - Two MySQL databases
